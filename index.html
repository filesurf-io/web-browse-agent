Web Browse Agent

AI-powered web browsing and UI testing agent with Playwright integration. Written in Go.

View on GitHub Visit FileSurf

Quick start

git clone https://github.com/filesurf-io/web-browse-agent.git
cd web-browse-agent
go build -o web-browse-agent ./cmd/agent
./web-browse-agent "go to google.com and search for playwright"

Features

  • Agentic Loop — recursive tool execution with automatic retry
  • Playwright Integration — full browser automation
  • Multi-LLM — OpenAI GPT-4 and Anthropic Claude support
  • 18 Browser Tools — click, type, screenshot, evaluate, tabs, and more
