Web Browse Agent

A sessionful browser automation CLI for persistent web browsing sessions. Written in Go.

View on GitHub Visit FileSurf

Quick start

git clone https://github.com/filesurf-io/web-browse-agent.git
cd web-browse-agent
make build
make install-deps
web_browse_agent --session test open https://example.com

Features

  • Persistent Sessions — browser state maintained across commands
  • Full Browser Automation — powered by Playwright
  • Stealth Mode — Cloudflare/bot-detection bypass built-in
  • JSON Output — machine-readable with --json
